One component, every provider

The card below composes six dom-provider-* packages in a single providers: [...] array — all typed onto this:

  • this.i18n — translations (EN / DE / AR), params from templates
  • this.direction — LTR / RTL (auto-follows the locale)
  • this.colorScheme — light / dark / auto
  • this.a11y — screen-reader announcements + dev CSS audit
  • this.logger — a scoped @youneed/logger child
  • this.store — a Zustand store (selector-gated re-renders)

Open the devtools panel (launcher, bottom-right) — alongside the built-in Components / Time-travel / Styles tabs you'll find i18n, a11y and zustand tabs, each fed by its capture plugin.
