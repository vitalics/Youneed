The framework

One paradigm, every layer.

Factory → base class → decorators. DOM component, HTTP controller, SSR page, CLI command — always the same mental model.

@youneed/dom
Custom Elements
Reactive components on native Shadow DOM. Signals, computed, tasks, schedulers — no virtual DOM.
@youneed/server
Typed HTTP
Schema validation, guards, HTTP/1.1+2+3, WebSocket, SSE. OpenAPI auto-generated.
@youneed/ssr
SSR + Pages
Declarative Shadow DOM. Speculation Rules, islands, SSG — all first-class.
@youneed/dom-router
SPA Router
Hash, history, query routing modes. Zero deps, tiny bundle.
@youneed/cli
CLI Framework
Terminal apps with 15+ middleware: prompts, hotkeys, progress, i18n, screen.
@youneed/test
Test Framework
Class + decorator tests, same paradigm. Snapshots, resilience, 5 reporter formats.
@youneed/devtools
DevTools
Floating inspector with component tree, time-travel debugger, scheduler swap.
@youneed/orm · kv
ORM + Storage
SQL + NoSQL ORM (MySQL, MongoDB). KV store with Redis adapter.
@youneed/schema
Schema + Validation
Type-safe schemas with OpenAPI/AsyncAPI generation. Used across all layers.
@youneed/logger
Logging
Structured logger with pluggable transports (stdout, file, HTTP) and Datadog, i18n plugins.
@youneed/i18n · dom-provider-a11y
i18n & Accessibility
Internationalization utilities + a11y context provider for DOM components and CLI apps.
@youneed/vite-plugin
Vite Integration
Lowers TC39 decorators in Vite builds via esbuild. Works alongside React, Vue, and Svelte plugins.
@youneed/ts-plugin
TS Language Plugin
TypeScript language service plugin — editor completions and diagnostics for decorators and templates.
new
@youneed/ai-skill
AI Skills
LLM/AI integration skill — wire language models into your server controllers and CLI commands with the same decorator API.

Quick start

Up and running in 3 steps.

Works with tsx out of the box. Add @youneed/vite-plugin for Vite builds.

1
Install
terminal
# Pick your packages
pnpm add @youneed/dom
pnpm add @youneed/server
pnpm add @youneed/ssr

# Vite support
pnpm add -D @youneed/vite-plugin
2
Create a component
app.ts
import { Component, html }
  from "@youneed/dom";

@Component.define()
class MyApp
  extends Component("my-app") {

  count = this.signal(0);

  render() {
    return html`<button
      @click=\${() =>
        this.count.update(n => n+1)
      }>\${this.count()}</button>`;
  }
}
3
Start the server
server.ts
import { Application,
  Controller }
  from "@youneed/server";

class Api
  extends Controller("/api") {

  @Controller.get("/ping")
  ping() {
    return { ok: true };
  }
}

Application(Api)
  .listen(3000);
$ tsx server.ts → http://localhost:3000
